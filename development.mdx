<Info>
  **Prerequisite**: Please install Python (version 3.12 or higher) before proceeding.
</Info>

Follow these steps to install and run cointracker-cbe on your platform:

**Step 1**: Install CBE:

<CodeGroup>

```bash pip
pip install cointracker-cbe
```


```bash yarn
yarn global add cointracker-cbe
```

</CodeGroup>

**Step 2**: Navigate to the docs directory (where the `docs.json` file is located) and execute the following command:

```bash
cointracker-cbe dev
```

A local preview of your cost basis engine will be available at `http://localhost:3000`.

## CBE Versions

If your local website doesn't align with the production version, please update the CLI:

<CodeGroup>

```bash pip
pip install cointracker-cbe@latest
```


```bash yarn
yarn global upgrade cointracker-cbe
```

</CodeGroup>

## Validating Features

The CLI can assist with validating features used in your platform. To identify any issues, use the following command:

```bash
cointracker-cbe diagnostics
```

## Deployment

<Tip>

</Tip>

Please work with your account manager.

## Troubleshooting

<AccordionGroup>
  <Accordion title='Error: Could not load the "tax lots" module'>
    This may be due to not having any imported transactions, or not having a complete cost basis run

    1. Add transactions into the CBE platform
    2. Make sure Cost Basis Job runs and completes at least once
  </Accordion>
  <Accordion title="Issue: Encountering an unknown error during development">
    Solution: Go to the root of your device and delete the \~/.cbe-temp folder. Afterwards, run `cointracker-cbe dev` again.
  </Accordion>
</AccordionGroup>
